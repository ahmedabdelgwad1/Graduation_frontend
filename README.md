# توثيق مشروع متحف الآثار بمكتبة الإسكندرية (Bibliotheca Alexandrina Museum)

هذا الملف يحتوي على شرح مفصل لهيكلية المشروع، التقنيات المستخدمة، وكيفية عمل الواجهات والتنقل بينها.

---

## 🚀 التقنيات المستخدمة (Tech Stack)
تم بناء هذا المشروع بأحدث معايير تطوير واجهات الويب:
- **إطار العمل الأساسي:** [Next.js 14/15 (App Router)](https://nextjs.org/) مع **React 19**.
- **لغة البرمجة:** [TypeScript](https://www.typescriptlang.org/) لضمان أمان الأنواع وتقليل الأخطاء.
- **التصميم وتنسيق العناصر:** [Tailwind CSS v4](https://tailwindcss.com/) مع استخدام (CSS Variables) للألوان لتسهيل التحكم في النمط الليلي والذهبي للمتحف.
- **إدارة الحالة (State Management):** [Zustand](https://zustand-demo.pmnd.rs/) لإدارة اللغة الحالية والمقاطع المختارة بسلاسة دون تعقيد.
- **الحركات والأنيميشن:** [Framer Motion](https://www.framer.com/motion/) لإضافة حركات انتقال (Page Transitions) وتأثيرات (Hover/Click) احترافية.
- **الأيقونات:** [Lucide React](https://lucide.dev/) لأيقونات الواجهة الرسومية (مثل العدسة، المكبر، الميكروفون، والمرشد الذكي).

## 🎨 الهوية البصرية والتصميم (Aesthetic & Design)

- **النمط (Theme):** "المتحف ليلاً" - يعتمد على دمج اللون الأسود الكحلي الداكن مع لمسات من اللون الذهبي الملكي المضيء.
- **تأثير ورق البردي (Papyrus):** تم إضافة فلاتر بصرية SVG لخلفية التطبيق لتعطي إحساس ورقة البردي المصرية القديمة (تجده معرفاً في `globals.css` كـ `papyrus-texture`).
- **الخطوط:** 
  - `Cinzel`: خط إنجليزي أنيق وعريق يُستخدم للعناوين الجذابة.
  - `Noto Naskh Arabic`: خط عربي كلاسيكي مخصص للنصوص العربية.
  - `Cormorant Garamond`: خط إنجليزي للنصوص الوصفية يعطي طابعاً تاريخياً.

## 🌐 دعم اللغات (Internationalization - i18n)

المشروع يدعم اللغتين العربية والإنجليزية بشكل كامل. 
- يتم الاعتماد على الـ URL لتحديد اللغة (مثال: `/ar/welcome` أو `/en/welcome`).
- المكون `[locale]/layout.tsx` يقوم بتحديد اتجاه الصفحة `dir="rtl"` إذا كانت اللغة عربية، و `dir="ltr"` للإنجليزية.
- دالة `getDictionary` تقوم بقراءة ملف الـ JSON المطابق واستخراج النصوص ليتم حقنها داخل المكونات.

## 📱 وصف الشاشات (Screens)

### 1. شاشة الترحيب (Welcome Screen)
- توفر دخولاً هادئاً للمتحف عبر زرين (العربية / English). 
- تستخدم مؤثرات زجاجية (Glassmorphism) ولمعان خفيف عند التأشير.
- تقوم بتخزين اختيار المستخدم في (Zustand Store) ثم توجيهه إلى صفحة المجموعات `/sections`.

### 2. صفحة المجموعات (Our Collections)
- تعرض أقسام المتحف المتنوعة (الآثار المصرية القديمة، الآثار اليونانية، الآثار الإسلامية، إلخ).
- يتم تمثيل كل قسم بـ `SectionCard` يحوي أيقونة مناسبة.

### 3. صفحة القطع الأثرية (Artifacts Grid)
- تستعرض القطع الأثرية بداخل المجموعة التي تم اختيارها.
- يوجد شريط للبحث عن القطع.
- في حالة غياب صورة القطعة الأثرية الحقيقية، تم توفير `Fallback UI` على شكل شعار ذهبي مزخرف.

### 4. صفحة تفاصيل القطعة والمحادثة (Artifact Detail & Chat)
تنقسم في شاشات الحواسيب (Desktop) إلى قسمين رئيسيين:
- **اليمين / اليسار (حسب اتجاه اللغة):** عرض التفاصيل التاريخية للقطعة المحددة واسم القاعة، وصورتها، ووصفها.
- **لوحة المحادثة (Chat Panel):** محادثة حية مع مرشد المتحف الذكي `Alex` متصلة برمجياً بواجهة خلفية (Backend). توفر أسئلة سريعة (Quick Questions) لكسر الجليد.

## 💻 كيفية التشغيل محلياً للمطورين

**1. تنزيل التحديثات (إن لزم الأمر):**
```bash
npm install
```

**2. تشغيل بيئة التطوير (Development Server):**
```bash
npm run dev
```

**3. بناء نسخة الإنتاج (Production Build):**
```bash
npm run build
npm start
```
